# Verstka
Чек-лист для вёрстки блоков «Яндекс Маршрутов»
В рамках проекта был разработан и внедрён чек-лист для проверки вёрстки формы бронирования и элементов навигационной карты в сервисе «Яндекс Маршруты».

Исследуемый набор данных и объекты исследования
Объектом исследования стали ключевые интерфейсные блоки: форма бронирования и элементы навигационной карты. Для анализа использовались макеты и технические требования, на основе которых был сформирован перечень критериев для проверки корректности и единообразия вёрстки.

Основные пункты и цели исследования
Составление структурированного чек-листа для валидации вёрстки.
Проверка соответствия реализации макетам и требованиям.
Выявление и минимизация визуальных и функциональных ошибок.
Обеспечение единообразия отображения интерфейса на разных устройствах.
Результат
Внедрение чек-листа позволило повысить качество вёрстки, обеспечить её единообразие и снизить риск появления ошибок, что положительно сказалось на пользовательском опыте.

Используемые инструменты
Чек-лист для валидации.
Ручная проверка вёрстки.
